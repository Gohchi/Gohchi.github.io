Welcome to my mini korean trainer app.

Instructions

Numbers
  • Press NEXT NUMBER to get a random number based on the number type.
    (when pressing on the number twice, a prompt will let you edit the value)
  • Press SHOW DETAILS to show the number's name in hangeul.
    (when pressing on the text, each syllable will be divided into buttons to hear each pronunciation)
  • Press PRONUNCIATION to hear the pronunciation.
More options
Press MORE button to show the advanced menu. On this menu you can change the following options.
  • Practice type: You can change between numbers and vocabulary*.
  • Voice type (experimental): You can change between two types of synthesized voices.
  • Voice rate: You can change the voice speed.
  • Number type: You can change the number type between Native Korean(1) and Sino-Korean(2).

* not implemented yet.

(1) the numbers shown are from 1-12

(2) the numbers shown are from 1-9999

Notes

2020-07-04 I started practicing the Korean language 6 weeks ago. It was difficult for me to memorize the numbers, so I started to create a simple application to get random numbers. The first thing I noticed was that I had to search my notes every time I forgot the name of a number.
So I thought, if I take the number that's already on the screen and use a dictionary with the names, I could create the full name.
Later I realized that sometimes I forgot the pronunciation. It was then that I found the tool to synthesize the voice.
